## 简介 🌟

这是一个用c语言编写的基于SDL库的俄罗斯方块

## 功能 🚀

- 可以调节音量
- 可以调节方块下落速度
- 有方块下落预览模式
- `tab` 键可以切换隐藏模式和显示模式
- `Esc` 键暂停游戏，可以保存进度，也可以回退一步

## 使用方法 📘

可以使用 vscode 编译成 exe 可执行文件运行
